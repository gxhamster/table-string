# Table String
> Table string is a library created to ease the use of strings in c
> It is made to be performant and is heavily inspired 

![](header.png)

## Installation

OS X & Linux:

```sh
cmake .
make 
./table_str
```
Windows:

```sh
cmake . -G "MinGW Makefiles"
make
table_str.exe
```
